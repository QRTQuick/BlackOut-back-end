API Endpoints

POST /api/convert
Convert files between 50+ formats. Upload any supported file and specify target format. 
curl -X POST "https://nodeblack.onrender.com/api/convert?target_format=jpg" \
  -H "X-API-Key: blackout-secret-key" \
  -F "file=@image.png"
GET /api/download/{task_id}
Download converted file using task ID. Files expire after 10 minutes. 
curl -O "https://nodeblack.onrender.com/api/download/abc123-def456"
GET /api/status/{task_id}
Check conversion status. Returns processing, ready, failed, or expired. 
curl "https://nodeblack.onrender.com/api/status/abc123-def456"
GET /api/files
List all converted files in database with status and download URLs. 
curl "https://nodeblack.onrender.com/api/files"
GET /api/formats
Get all supported file formats and available conversions. 
curl "https://nodeblack.onrender.com/api/formats"
GET /api/test
Health check endpoint. Verify API is running and storage is accessible. 
curl "https://nodeblack.onrender.com/api/test"
GET /api/ping
Keep-alive ping endpoint. Used internally to prevent server sleep. 
curl "https://nodeblack.onrender.com/api/ping"
GET /docs
Interactive API documentation with Swagger UI. Test endpoints directly. 
https://nodeblack.onrender.com/docs
